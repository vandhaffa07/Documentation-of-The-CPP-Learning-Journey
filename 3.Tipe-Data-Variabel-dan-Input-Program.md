# TIPE DATA, VARIABEL, DAN INPUT PROGRAM
Dalam pemrograman, kita tidak hanya sekadar menampilkan teks statis, tetapi juga mengelola informasi. Untuk melakukan hal tersebut, kita memerlukan tempat penyimpanan data yang disebut Variabel, aturan jenis data yang disebut Tipe Data, dan cara mengambil data dari pengguna yang disebut Input. Memahami ketiga hal ini adalah fondasi utama sebelum membangun logika program yang lebih kompleks.

## VARIABEL
Variabel adalah sebuah nama atau identifier yang mewakili suatu lokasi di memori komputer (RAM) dan digunakan untuk menyimpan nilai. Variabel bisa diibaratkan sebagai sebuah kotak yang memiliki label nama, dimana kita bisa dengan sesuka hati memanipulasi isi dari kotak tersebut seperti memasukkan sesuatu ke dalam kotak tersebut, mengambilnya, atau menggantinya dengan isi yang baru selama program berjalan. 

Dalam C++, sebelum menggunakan variabel, kita wajib terlebih dahulu melakukan deklarasi dengan tipe data tertentu (misalnya int, char, long). Deklarasi sendiri adalah pernyataan/pemberitahuan yang digunakan untuk memberi tahu compiler mengenai keberadaan sebuah entitas (dalam kasus ini adalah variabel) dengan tujuan untuk memesan memori dan memastikan program mengenali identitas tersebut. 

Berikut adalah contoh file program deklarasi dalam C++ :
```bash
nano program.cpp
```
```cpp
#include <iostream>
using namespace std;

int main(){
  int umur;
  double beratbadan;
  char inisial;
return 0;
}
```
Pada titik ini, penting untuk dipahami bahwa deklarasi di atas belum mengisi variabel dengan nilai apa pun. Kita hanya memberi pernyataan ke compiler bahwa terdapat variabel bernama umur dengan tipe data integer, beratbadan dengan tipe data double, dan inisial dengan tipe data character, sehingga compiler akan menganggap kode tersebut sebagai valid dan mulai mengalokasikan ruang di memori yang. Hal ini diperjelas lagi ketika kita melakukan compile dan menjalankan program tersebut :
```bash
g++ program.cpp -o program
```
```bash
./program
```
Dapat diperhatikan bahwa ketika program telah dicompile dan dijalankan, program akan berjalan tanpa adanya indikasi error walaupun tidak menampilkan output apa pun di layar. Hal ini wajar dikarenakan kode yang kita tulis hanya berfokus pada pernyataan dan pemesanan memori (deklarasi), bukan pada instruksi untuk menampilkan data (output).

Namun, apa yang terjadi jika kita memaksa program untuk menampilkan isi dari variabel yang baru saja dideklarasikan tersebut menggunakan perintah cout? program mungkin tetap bisa berjalan, tetapi hasil yang muncul di layar akan menjadi angka yang sangat acak atau yang biasanya disebut sebagai nilai sampah (garbage value). Seacar teknis ketika kita mendeklarasikan variabel, misalnya `int umur;`, sistem operasi hanya mengalokasikan satu petak memori untuk kita. Masalahnya, memori tersebut tidak dibersihkan terlebih dahulu. Memori tersebut kemungkinan besar masih berisi sisa-sisa data dari program lain yang baru saja ditutup. sehingga jika kita langsung memanggilnya dengan cout, maka yang terjadi adalah seperti ini :
```bash
nano program.cpp
```
```cpp
#include <iostream>
using namespace std;

int main(){
  int umur;
  double beratbadan;
  char inisial;
cout<<umur<<endl;
return 0;
}
```
```bash
g++ program.cpp -o program
```
```bash
./program
32402
```
```bash
./program
29805
```
```bash
./program
30611
```
Terlihat bahwa setiap kali kita menjalankan program tersebut, nilai dari variabel umur selalu berubah-ubah, dan inilah yang disebut dengan nilai sampah (garbage value)

Untuk mencegah munculnya nilai sampah (garbage value) yang tidak terduga, kita perlu melakukan proses yang disebut dengan Inisialisasi. Inisialisasi adalah proses memberikan nilai awal ke dalam variabel pada saat variabel tersebut dideklarasikan. Dengan melakukan inisialisasi, kita memastikan bahwa petak memori yang kita pesan sudah dibersihkan dari sisa-sisa data program lain dan diisi dengan nilai yang kita inginkan.

Cara yang paling umum digunakan untuk melakukan inisialisai adalah dengan menggunakan tanda `=` seperti ini :
```cpp
int umur = 15;
double beratbadan = 55.5;
char inisial = 'I';
```
Secara teknis, penggunaan tanda `=` saat deklarasi disebut sebagai copy-initialization dengan mekanisme menyalin nilai di sebelah kanan operator ke dalam lokasi memori variabel di sebelah kirinya.

Berikut adalah perbandingan file program jika kita menggunakan inisialisasi:
```bash
nano program.cpp
```
```cpp
#include <iostream>
using namespace std;

int main(){
  int umur = 15;
  double beratbadan = 55.5;
  char inisial = 'I';

  cout << "Umur: " << umur << endl;
  cout << "Berat Badan: " << beratbadan << " kg" << endl;
  cout << "Inisial: " << inisial << endl;

return 0;
}
```
```bash
g++ program.cpp -o program
```
```bash
./program
Umur: 15
Berat Badan: 55.5 kg
Inisial: I
```
Terlihat bahwa ketika program dijalankan, nilai yang muncul di layar sepenuhnya konsisten dan sesuai dengan angka yang kita masukkan di dalam kode dan tidak ada lagi angka acak atau sampah yang muncul. Hal ini mengartikan bahwa proses inisialisai dan penghapusan sisa-sisa data lama yang mungkin tertinggal di alamat memori berlangsung dengan lancar

## TIPE DATA
Dari tadi kita telah menyebutkan istilah Tipe Data berkali-kali saat mendeklarasikan variabel, seperti int, double, dan char. Namun, apa sebenarnya fungsi dan maksud dari kata-kata tersebut? Tipe data adalah perwujudan klasifikasi yang menentukan jenis, ukuran, dan format nilai yang dapat disimpan dalam sebuah variabel. Tipe data memberi tahu compiler bagaimana cara mengolah data yang ada di dalam memori, apakah data itu angka, teks, atau logika. Sederhananya, jika variabel adalah sebuah kotak, maka tipe data adalah instruksi yang menentukan seberapa besar kotak tersebut dan jenis barang apa yang boleh masuk ke dalamnya. 

Secara umum, dalam C++ terdapat beberapa tipe data primitif yang sering digunakan sesuai dengan kebutuhan datanya:
- **Short, Int, & Long**: Digunakan untuk bilangan bulat.
- **Float & Double**: Digunakan untuk bilangan desimal/pecahan
- **Char**: Digunakan untuk menyimpan satu karakter tunggal (seperti 'A', '7', atau '$').
- **Bool**: Tipe data logika yang hanya memiliki dua kemungkinan nilai yakni true (1) atau false (0).

Untuk lebih rincinya, kita bisa melihat bagaimana sistem komputer mengalokasikan memori dan menentukan batas maksimum serta minimum dari setiap tipe data tersebut. Kita dapat menggunakan fungsi sizeof() untuk melihat ukuran memori (dalam satuan Byte) dan library <limits> untuk membedah kapasitas tampungnya secara teknis melalui program berikut:

```bash
nano cek_tipe.cpp
```
```cpp
#include <iostream>
#include <limits>

int main(){

        int a;
        long b;
        short c;
        float d;
        double e;
        char f;
        bool g;

        std::cout<<"int"<<std::endl;
        std::cout<<"Ukuran : "<<sizeof(a)<<" Byte"<<std::endl;
        std::cout<<"Nilai Maksimum : "<<std::numeric_limits<int>::max()<<std::endl;
        std::cout<<"Nilai Minimum : "<<std::numeric_limits<int>::min()<<std::endl;

        std::cout<<"\nlong"<<std::endl;
        std::cout<<"Ukuran : "<<sizeof(b)<<" Byte"<<std::endl;
        std::cout<<"Nilai Maksimum : "<<std::numeric_limits<long>::max()<<std::endl;
        std::cout<<"Nilai Minimum : "<<std::numeric_limits<long>::min()<<std::endl;

        std::cout<<"\nshort"<<std::endl;
        std::cout<<"Ukuran : "<<sizeof(c)<<" Byte"<<std::endl;
        std::cout<<"Nilai Maksimum : "<<std::numeric_limits<short>::max()<<std::endl;
        std::cout<<"Nilai Minimum : "<<std::numeric_limits<short>::min()<<std::endl;

        std::cout<<"\nfloat"<<std::endl;
        std::cout<<"Ukuran : "<<sizeof(d)<<" Byte"<<std::endl;
        std::cout<<"Nilai Maksimum : "<<std::numeric_limits<float>::max()<<std::endl;
        std::cout<<"Nilai Minimum : "<<std::numeric_limits<float>::min()<<std::endl;

        std::cout<<"\ndouble"<<std::endl;
        std::cout<<"Ukuran : "<<sizeof(e)<<" Byte"<<std::endl;
        std::cout<<"Nilai Maksimum : "<<std::numeric_limits<double>::max()<<std::endl;
        std::cout<<"Nilai Minimum : "<<std::numeric_limits<double>::min()<<std::endl;

        std::cout<<"\nchar"<<std::endl;
        std::cout<<"Ukuran : "<<sizeof(f)<<" Byte"<<std::endl;
        std::cout<<"Nilai Maksimum : "<<std::numeric_limits<char>::max()<<std::endl;
        std::cout<<"Nilai Minimum : "<<std::numeric_limits<char>::min()<<std::endl;

        std::cout<<"\nbool"<<std::endl;
        std::cout<<"Ukuran : "<<sizeof(g)<<" Byte"<<std::endl;
        std::cout<<"Nilai Maksimum : "<<std::numeric_limits<bool>::max()<<std::endl;
        std::cout<<"Nilai Minimum : "<<std::numeric_limits<bool>::min()<<std::endl;
        return 0;
}
```

Sampai di titik ini, kita sudah mengetahui bahwa tipe data berperan sebagai aturan/instruksi logis bagi suatu variabel. Namun, apa yang terjadi jika kita mencoba mendeklarasikan atau menginisialisasi variabel tanpa menyertakan tipe data sama sekali? C++ adalah bahasa pemrograman dengan aturan statically typed, yang artinya setiap variabel wajib diketahui tipe datanya pada saat kompilasi. Jika kita mencoba mendeklarasikan atau menginisialisasi variabel tanpa menyertakan tipe datanya maka program tidak akan bisa berjalan sama sekali dan menampilkan error ditengah-tengah fase compiling.

Sebagai contoh, mari kita simulasikan program sederhana yang digunakan untuk menampilkan nilai dari variabel seperti ini :
```bash
nano test.cpp
```
```cpp
#include <iostream>
using namespace std;

int main() {
    angka = 10; // Mencoba membuat variabel tanpa tipe data
    cout<<angka<<endl;
    return 0;
}
```
```bash
g++ test.cpp -o test
error: ‘angka’ was not declared in this scope
```
Dapat terlihat bahwa compiler akan berhenti dan memberikan pesan error seperti'angka' was not declared in this scope. Hal ini disebabkan karena compiler tidak mengetahui tipe data variabel tersebut, sehingga ia tidak bisa menentukan berapa byte memori yang harus dia alokasikan dan bagaimana cara menerjemahkan biner di dalam RAM tersebut menjadi nilai yang manusia pahami.

Selain itu, permasalahan lain yang sering terjadi bukan hanya pada ketiadaan tipe data, tetapi juga pada ketidaksesuaian antara tipe data dan nilai yang diberikan saat inisialisasi. Dalam hal ini terdapat beberapa kasus yang sering terjadi, diantarnya :
- **Narrowing Conversion** : Jika kita memasukkan angka desimal ke dalam tipe data bilangan bulat (int), C++ tidak akan memberikan indikasi error, melainkan akan membuang angka di belakang koma secara paksa.
```bash
nano test.cpp
```
```cpp
#include <iostream>
using namespace std;

int main(){
  int nilai = 95.5;
  cout<<nilai<<endl;
  return 0;
}
```
```bash
g++ test.cpp -o test
```
```bash
./test
95
```

- **Kasus Ketidaksesuaian Karakter dan Angka** : Jika kita memberikan nilai berupa karakter ke dalam tipe data int, C++ akan mengambil nilai ASCII (kode angka) dari karakter tersebut. Sebaliknya, jika kita memasukkan angka ke dalam tipe data char, ia pun akan berubah menjadi simbol.
```bash
nano test.cpp
```
```cpp
#include <iostream>
using namespace std;

int main(){
  int huruf = 'a';
  cout<<huruf<<endl;
  return 0;
}
```
```bash
g++ test.cpp -o test
```
```bash
./test
97
```
Begitu pula sebaliknya :
```bash
nano test.cpp
```
```cpp
#include <iostream>
using namespace std;

int main(){
  char skor = 97;
  cout<<skor<<endl;
  return 0;
}
```
```bash
g++ test.cpp -o test
```
```bash
./test
a
```

- **Overflow** : Jika kita memasukkan nilai yang lebih besar dari batas maksimum tipe data tersebut, akan terjadi fenomena Overflow, dimana nilai yang disimpan tidak lagi merepresentasikan angka yang sebenarnya, melainkan berputar kembali ke rentang nilai minimum yang dapat ditampung oleh tipe data tersebut.
```bash
nano test.cpp
```
```cpp
#include <iostream>
using namespace std;

int main(){
  short angka = 32767; // Batas maksimum short
  angka = angka + 1;
  cout <<angka<<endl;
  return 0;
}
```
```bash
g++ test.cpp -o test
```
```bash
./test
-32768
```
