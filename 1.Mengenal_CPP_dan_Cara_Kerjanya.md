# MENGENAL C++ DAN CARA KERJANYA

## APA ITU C++?
C++ adalah bahasa pemrograman serbaguna yang dikembangkan oleh ilmuwan komputer Denmark, Bjarne Stroustup pada awal 1980-an sebagai perluasan dari bahasa pemrograman C dengan menambahkan beberapa fitur-fitur canggih, seperti pemrograman berorientasi objek (OOP) yang meliputi class, inheritance, polymorphism, encapsulation, dll. C++ juga terkenal dengan performanya yang optimal dengan menawarkan kontrol yang lebih detail terhadap manajemen memori sehingga programmer dapat menulis kode yang berjalan dengan sangat efisien. Hingga saat ini, C++ terus menerima pembaruan secara rutin. Maka, ketika terdapat seseorang yang mengatakan bahwa C++ adalah bahasa pemrograman yang kuno, ketahuilah bahwa mereka menerima pemahaman yang salah, karena faktanya C++ adalah bahasa pemrograman yang tetap relevan dan diminati oleh para pakar IT di dunia hingga saat ini.

## FUNGSI DAN KEGUNAAN C++
### 1. Pengembangan Sistem Operasi
C++ seringkali digunakan dalam pengembangan sistem operasi seperti Windows, Linux, dan MacOS karena kemampuannya dalam mengakses memori secara langsung dan berinteraksi dekat dengan hardware. Pada sistem operasi, komponen seperti kernel, memory management, dan driver perangkat keras membutuhkan kontrol penuh terhadap resource. Berbeda dengan bahasa pemrograman tingkat tinggi yang terlalu abstrak, C++ memberikan akses langsung ke memori melalui pointer tanpa menghilangkan struktur pemrograman yang terorganisir. Dengan kata lain, C++ memungkinkan developer bekerja cukup dekat dengan hardware tanpa harus turun sepenuhnya ke assembly.

### 2. Game Development
Dalam pengembangan game, C++ banyak digunakan terutama sebagai bagian dari game engine dan logika inti. Sebagian besar game berat seperti AAA membutuhkan performa real-time, di mana setiap frame harus diproses dalam waktu yang sangat singkat. Yang artinya, keterlambatan sedikit saja pada proses rendering dapat menyebabkan lag atau penurunan kualitas pengalaman pengguna. Maka dari itu, C++ dipilih karena menawarkan performa eksekusi yang tinggi. C++ merupakan salah satu bahasa yang terkompilasi(compiled language) di mana kode sumber C++ langsung diterjemahkan sepenuhnya ke dalam kode mesin oleh kompiler sebelum dieksekusi. Hal ini membuat proses  eksekusi menjadi sangat cepat dan responsif, sehingga sangat cocok untuk game yang membutuhkan rendering grafis tinggi secara real time.

### 3. Embedded System dan IOT
C++ adalah bahasa pemrograman yang sangat populer dan cocok untuk Embedded Systems dan Internet of Things (IoT) karena kombinasi antara efisiensi tinggi, kontrol tingkat rendah, dan fitur pemodelan berorientasi objek yang kuat. C++ memungkinkan pengembang untuk berinteraksi langsung dengan perangkat keras seperti MCU, sensor, dan memori sehingga memberikan pengembang kendali penuh atas komponen/modul hardware yang ada. Kode C++ pun umumnya memiliki ukuran biner yang kecil, sehingga cocok untuk perangkat IoT berdaya rendah. Selain itu, C++ juga mendukung pemrograman berorientasi objek yang memungkinkan pengembang mengatur kode menjadi komponen yang modular dan mudah dikelola.

## ALUR KERJA PROGRAM C++
Sebuah program C++ tidak langsung dijalankan begitu saja setelah ditulis. Terdapat beberapa tahapan yang dilalui sebelum program dapat dieksekusi oleh sistem. Secara umum, alurnya adalah sebagai berikut:

### 1. Preprocessing
Tahap preprocessing pada kompilasi C++ adalah langkah pertama sebelum source code dikompilasi menjadi bahasa mesin. Pada tahap ini, program yang disebut preprocessor akan memindai kode sumber dan menangani direktif-direktif khusus yang ditandai dengan simbol '#'. Hasil dari tahap ini adalah kode sumber yang telah diperluas. Direktif-direktif khusus yang telah kita tulis seperti `#include <filename>` dan `#define` akan diproses oleh preprocessor dengan menyisipkan isi file header (#include) yang disertakan serta mengganti macro (#define) dengan nilai yang sesuai. Proses-proses ini kemudian digabungkan dengan source code yang kita miliki menjadi suatu file baru dengan ekstensi .i
